# 🚚 Supply Chain Analytics Dashboard

## 📌 Overview
This Power BI dashboard provides insights into supply chain performance by tracking key metrics such as revenue, orders, inventory levels, delivery performance, and supplier efficiency.

## 🎯 Objectives
- Monitor overall supply chain performance
- Track revenue and order trends
- Analyze OTIF (On-Time In-Full) performance
- Identify delivery delays and inventory issues
- Evaluate supplier performance

## 📊 Key Metrics
- Total Revenue
- Total Orders
- OTIF %
- Average Delivery Delay
- Inventory Stock Levels
- Supplier Performance
- Revenue by Product Category
- Regional Revenue Distribution

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel

## 📈 Dashboard Features
- Interactive Filters and Slicers
- KPI Cards
- Revenue Trend Analysis
- Category-wise Performance Analysis
- Regional Insights
- Supplier Performance Monitoring

## 🔍 Business Insights
- Identified top-performing product categories.
- Tracked delivery delays impacting customer satisfaction.
- Analyzed regional revenue contribution.
- Evaluated supplier performance using operational KPIs.
